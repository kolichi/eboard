Product PDF Annotation Tool

Enhance Product PDFs with Dynamic Annotations and Descriptions for a Seamless User Experience.

The 'Product PDF Annotation Tool' enhances your product by enabling dynamic annotations on product PDFs. Administrators can easily add annotations with detailed descriptions to specific parts of the product PDF, allowing users to review these annotations on the website. This tool provides a seamless experience for managing and presenting product information.

Product PDF Annotation Tool

Installing the Product PDF Annotation Tool add-on in Odoo.

To install the Product PDF Annotation Tool module along with custom addons, start by accessing the Odoo Apps menu. Next, clear any existing filters from the search field. Then, search for the "Product PDF Annotation Tool" addons. Once you locate the desired addon, proceed by selecting it to begin the installation process.

Steps to Upload and Annotate Product Catalog PDFs.

After installing the module, open an existing or new product and click the "Documents" smart button at the top.

On the Product Documents screen, upload the product catalog PDF that you want to display on the website. After uploading the PDF, set the "Show on Product Page" option to true. Once enabled, a new button labeled "PDF Annotation" will appear. Click this button to be redirected to the PDF annotation editor screen.

PDF Annotation Editor.

Now you are on the PDF annotation editor screen, where you can see the current PDF with the pager information on the left side and the descriptions on the right side.

To start annotating your PDF, simply click on the PDF where you want to place a marker, which will then correspond to the description boxes.

Efficient PDF Annotation Interface with User-Friendly Features.

As shown in the screenshot below, you can easily add annotation markers on any page and provide corresponding descriptions. The interface is user-friendly, allowing direct access to any page from the pager. Clicking on a marker or its description will automatically highlight and scroll to the related item. You can also delete markers and descriptions using the delete icon. Markers can be dragged to better positions for precise placement. Multiple markers can be added across various PDFs as needed.

Annotation Integration: Preview Annotated PDFs on Website.

Upon completing PDF Annotation in the backend, a new button labeled "Preview: PDF" will appear on the product website page. Clicking this button allows users to preview the annotated PDF.

Clicking on "Preview PDF" opens a popup displaying the annotated PDF in read-only mode, allowing users to review detailed annotations for each part alongside their descriptions. This feature supports annotating multiple PDFs for the same products.
